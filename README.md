# Pong Game by Luca

A simple HTML page with JavaScript for Pong Game by Luca Necakov as part of CS Grade 11 class.

To view, open `PongGamebyLuca.html` in a browser.

Instructions:
- Press "SPACE" key to start the game
- Press "w" key to move Player1's pong up and "s" key to move player1's pong down
- Press "UP" key to move Player2's pong up and "DOWN" key to move Player2's pong down
- Soring happens when the ball goes out of bounds on the oponent's side
- When the ball goes out press "SPACE" key to start the next round
